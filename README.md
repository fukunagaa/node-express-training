# node-express-training
[webフレームワーク]expressを用いたサーバサイドのサンプル

## プロジェクト作成
- express-generator(管理者権限が必要)
```
$ npm install -g express-generator
```
- ディレクトリ作成(雛形)
```
$ express node-express-training
$ cd node-express-training
```

- インストール
```
$ npm install
```

- サーバ起動
```
$ npm run start
```

## 補足
